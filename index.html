My Work

Ink Tank

InkTank is an companion app that users in their journey from a desire for a tattoo until the final result marked into the skin. It’s inspirational in terms of styles, meanings, etc, and help identify the style of the tattoo. It will also help locate a tattoo studio and artist.

In this project I was the UX designer and did user research, competitive analysis, usability testing and low to high fidelity prototypes.

Perfect Properties

Real estate investment is an increasingly popular way for individuals to achieve financial security. The goal for this web app is to provide them with the expertise needed to get started. Perfect Properties is a responsive web app that provides property buyers with information on properties of interest.

In this project I was the UI designer for a responsive real estate web app. I designed the product user interface according to a brief from low to high fidelity prototypes, iconography, style guide and breakpoints..
